# Hawthorne-Dynamics-SP-Builder-Tools

SimplePlanes XML Editor
A modern XML editor for SimplePlanes, designed to make editing aircraft XML files faster, easier, and more organized.
This project was created to provide a clean, user-friendly interface that brings the most useful XML editing tools together in one place, eliminating the need to manually search through large XML files.
Features
📂 Open and save SimplePlanes XML files
⚙️ Edit common XML attributes
🔎 Find & Replace attribute values
🧩 Browse aircraft parts with filters
🎲 Rotation Randomizer
🛠️ Advanced editing tools
📊 File statistics
📝 Built-in operation log
🎨 Multiple UI themes
⚡ Fast and lightweight interface
Current Version
This is the Free Edition of SimplePlanes XML Editor and includes the core tools needed for efficient XML editing.
A Pro Edition with additional features, advanced tools, and productivity improvements is currently in development and will be released in the future.
