# Hawthorne Dynamics SP XML Editor

A lightweight XML editor for SimplePlanes, designed to simplify aircraft XML editing with a clean interface and practical tools.

## Features

- XML attribute editing
- Parts Browser
- Find & Replace
- Rotation Randomizer
- Multiple UI themes
- Advanced editing tools

The Free Edition is currently available. A Pro Edition with additional features is planned for future releases.
